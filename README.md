## Weather App
A responsive weather application built using React, Vite, and Tailwind CSS that allows users to search for real-time weather information of any city.

## Features
-Search weather by city name
-Displays temperature and weather conditions
-Fast and responsive UI
-Styled using Tailwind CSS
-Built with modern React (Hooks)

## Tech Stack
React (v19)
Vite
Tailwind CSS
JavaScript (ES6+)

## Features
-Efficient state management using useState and useEffect
-Dynamic UI updates without page reload
-Handles empty input and invalid city names gracefully
-Includes loading states for better user experience
-Component-based architecture with reusable components
-Optimized production build using Vite
-Ready for deployment on Netlify and Vercel
